В ДЗ необходимо сделать 2 варианта вывода числа и кнопок увеличивающих/уменьшающих это число.

1 вариант - число выводится в теге
2 вариант - число выводится в инпуте

- для обоих вариантов необходимо настроить минимальный и максимальный порог значения
- число подтягивается из состояния
- при инициализации числу присваивается значение из пропса min

дополнительное задание - валидация пропсов
- реализована проверка типа данных полученных пропсов
- реализована установка значения пропсов по умолчанию для случая если пропс(ы) не будет получен в компоненте
